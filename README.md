# Hangman
Игра Виселица, написанная на C++ (ООП). Проект для экзамена.

■ Слово выбирается компьютером из списка слов.
■ Список слов находится в файле в зашифрованном виде.
■ По завершении игры на экран выводится статистика игры:
• количество времени;
• количество попыток;
• искомое слово;
• буквы игрока.
